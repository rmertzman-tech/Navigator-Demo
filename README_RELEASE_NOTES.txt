Navigator Twin v2.8.2 Stable Demo — Release Notes

This release packages the first complete stable agency-navigation loop for demonstration use.

Included capabilities:
- Twin conversation and Twin Journal
- Document Capsule upload/review/save workflow
- Field Scan creation and Field Map Dashboard
- Context Assembly
- Field-specific Move Cards
- Cross-Field Move Comparison
- Current Move selection
- Current Move Deliberation Plan
- Fictional demo data mode
- Help / Quick Start / Glossary
- Local TXT and JSON export
- JSON import / restore

Stable demo changes in v2.8.2:
- Updated stable demo version labels
- Added reviewer-oriented Start Here guidance
- Strengthened privacy/security language
- Added explicit roadmap notes for PWA, Private API, PRF-22 import, Applications & Coaches, and simulation/timeline layers
- Preserved non-PWA fallback behavior with ENABLE_PWA = false

This release is intended for a separate stable demo repository.
